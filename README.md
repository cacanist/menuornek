# Atölye Hamur - QR Menu

Modern, mobile-first QR Menu for Atölye Hamur.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

- **Menu Data:** Edit `data/menu.json` to update categories and items.
- **Styling:** Update `app/globals.css` for theme colors.
